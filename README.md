# ESAME-SESTO-SENSO

Three.js WebGPU example based on the linked particles VFX demo, packaged as a Vite app.

## Install

```bash
npm install
```

## Start

For a quicker launch after the first install, run:

```bash
npm start
```

This starts the Vite dev server.

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```
